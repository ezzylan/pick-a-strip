# Pick A Strip

A web app that lets you create digital paper strip choices for fun and easy decision-making.

## Features

-   Use up to 10 strips
-   Display strips in separate view for decision-making
-   Randomize strips order

## Tech Stack

-   React + TypeScript + Vite
-   HeroUI (TailwindCSS)
