# Pick A Strip

A simple way to let friends pick a random strip and make spontaneous decisions.

## Features

-   Use up to 10 strips
-   Display strips in separate view for decision-making
-   Randomize strips order

## Tech Stack

-   React + TypeScript + Vite
-   HeroUI (TailwindCSS)
